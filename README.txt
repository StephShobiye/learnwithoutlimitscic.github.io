# Learn Without Limits CIC – Website

This repository contains the public-facing website for Learn Without Limits CIC.

The website acts as the main hub for accessing information, guidance, events, and resources supporting families navigating the Additional Learning Needs (ALN) system in Wales.

## Ownership

This platform is developed and owned by Golau Insight Ltd.

Learn Without Limits CIC operates the platform under licence for public benefit delivery.

## Purpose

The website provides:

- Programme overview and structure
- Access to guidance and resources
- Event information and registration routes
- Public-facing insight and engagement

It forms part of a wider ecosystem designed to:

- Prevent escalation within the ALN system
- Bridge gaps in access to support
- Enable long-term progression for families and young people

## Status

This is a live and evolving system, continuously updated based on real-world community need and programme development.
